日期类

  • 获取当前时间: System.currentTime()
  • 转换为日期格式: SimpleDateFormat(“yyyy-MM-dd:HH-mm-ss”)

JSON 序列化 Java对象

  • 将JSON反序列化成多个Java对象: JSON.parseArray(String,Class) -> List
  • 将JSON反序列化成Java对象: JSON.parseObject(String text) -> JSONObject
  • 获取JSONObject 中某个key的数据 : getJSONObject(String key) -> JSONObject
  • 获取Json 中某个值: )先获取该值所属的JSONObject 2)再调用JSONObject.getString() -> String
